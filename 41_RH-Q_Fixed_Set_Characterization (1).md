## 1. Characterization

**Lemma (RHϘ‑5 — Fixed‑sector structure).**  
φ is Q‑fixed iff:
(i) φ is FE‑symmetric; (ii) ˆφ is supported on the critical line (via σ_HP); (iii) φ lies in the Weil‑positivity cone; (iv) Hadamard normalization holds.

**Proposition (RHϘ‑6 — Separation by tests).**  
The family of Q‑fixed tests separates zeros: if a zero ρ with Re ρ ≠ 1/2 existed, there would be a fixed test for which W(φ)>0, contradiction with fixed‑sector constraints.

## 2. No Off‑Line Zeros

**Theorem (RHϘ‑7 — Positivity forbids off‑line contribution).**
